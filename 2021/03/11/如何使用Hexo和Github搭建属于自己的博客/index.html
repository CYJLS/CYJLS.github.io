如何使用Hexo和Github搭建属于自己的博客

发表于2021-03-11|更新于2021-04-02|教程
|字数总计:2.1k|阅读时长:7分钟|阅读量:

前言

在刚搭建完自己的个人博客的时候，我就写了一篇《个人博客的成功部署》，这篇其实就是表达一个开心的心情。想要搭建属于自己的个人博客网站的这个想法在我心里已经存在很久了，然后由于种种原因一直都没有搭建成功。这次搭建成功也是在过年放假期间才完成的，当我完成部署，在浏览器输入自己的Github仓库地址时，看到成功的那一刻，真的是满心的欢喜。

写这篇文章呢，主要是给自己梳理一遍建站的步骤，然后把这些步骤记录下来（避免自己忘记）。搭建自己的个人博客主要分成两部分：安装Hexo和部署到服务器（Github)，部署到Github这应该是大多数人的做法，虽然访问起来会有些慢，不过个人感觉够用了。

安装Hexo

在安装Hexo框架之前，我们还需要进行Node.js和Git的安装。我们的Hexo框架是通过Git来实现安装的。

检测软件是否安装成功

将博客部署的服务器上（Github Page）

其实呢，完成了本地博客的搭建也可以说是已经完成了个人博客的搭建。但是，为了能够随时随地和让他人也能访问我们的博客，我们需要把我们自己的博客部署到服务器上。接下来，我以部署到Github为例子，来说说如何把我们的本地博客部署到服务器上。
